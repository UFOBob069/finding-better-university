# Finding Better University

A modern website for Finding Better University, built with Next.js and integrated with the Finding Better Podcast. This platform helps people build the "better" they seek in life—personally and professionally—by sharing real-life stories of those who have been there and done that.

## Features

- Modern, responsive design with Tailwind CSS
- Integrated Spotify podcast player
- Firebase-powered waitlist system
- Smooth scrolling navigation
- SEO optimized

## Sections

- Hero section with animated background
- About section highlighting the mission
- How It Works explaining the process
- Benefits showcasing value proposition
- Waitlist for early access
- Latest podcast episode embed
- Footer with important links

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Firebase/Firestore
- Spotify Web Embed

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/UFOBob069/finding-better-university.git
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env.local` file with your Firebase configuration:
```
NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_auth_domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_storage_bucket
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
```

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
